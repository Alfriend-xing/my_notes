### 创建对象
创建对象需要以下三步
- 声明 ：声明一个对象，包括对象名称和对象类型
- 实例化 ：使用关键字new来创建一个对象
- 初始化 ：使用new创建对象时，会调用构造方法初始化对象
```java
// 下面的语句将创建一个Puppy对象
Puppy myPuppy = new Puppy( "tommy" );
```
### 访问对象的变量和方法
```java
/* 实例化对象 */
ObjectReference = new Constructor();
/* 访问类中的变量 */
ObjectReference.variableName;
/* 访问类中的方法 */
ObjectReference.MethodName();
```
